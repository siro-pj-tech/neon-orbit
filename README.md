# 🚀 ネオン・オービット (Neon Orbit)

ワンアクションで遊べるシンプルなハイスピード障害物回避ブラウザゲーム。
暇な時間で手軽に遊ぼう。

## 🌌 概要
光り輝くネオンの軌道を駆け抜け、次々と迫りくる障害物を回避せよ！直感的な操作と、疾走感のあるビジュアルが特徴のアーケードスタイルゲームです。

## 🎮 遊び方
- **操作**: 画面クリック または スペースキー
- **ルール**: 自機の軌道を切り替えて、障害物に当たらないように進み続けます。
- **目標**: どこまで長く生き残れるか、ハイスコアを目指しましょう！

## ✨ 特徴
- **サイバーパンク・ビジュアル**: ネオンカラーを基調とした鮮やかなエフェクト。
- **シンプル操作**: 誰でもすぐに遊べるワンボタン（ワンクリック）アクション。
- **ブラウザ対応**: インストール不要。PCやスマホのブラウザで即座にプレイ可能。

## 🛠 使用技術
- **HTML5 / CSS3** (デザイン・レイアウト)
- **JavaScript** (ゲームロジック・描画)
